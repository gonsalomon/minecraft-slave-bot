# Minecraft Bot

Bot de Mineflayer para minería automatizada con control vía MCP.

## Setup

1. Clonar el repo
2. `npm install`
3. Copiar `.env.example` a `.env` y completar los valores
4. `node bot.js`

## Requisitos

- Node.js 18+
- Servidor Minecraft con online-mode=false
- Username del bot en la whitelist del servidor
